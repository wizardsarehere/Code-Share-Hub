## Getting Started

1. Clone the repository:
```bash
git clone https://github.com/wizardsarehere/Code-Share-Hub.git
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm run dev
```

## Building for Production

To create a production build:

```bash
npm run build
```
